# ultralight-rs

Handcrafted Rust API for ultralight. See examples.

## Raw bindings

`/ultralight-sys` contains the raw straight-out-of-bindgen bindings.

## Ultralight support

Latest version : 1.2.1.

## Building

Set `ULTRALIGHT_SDK` env var to where Ultralight SDK is installed.
